# 🩺 Health Condition Prediction Model

## 📌 Project Overview

The **Health Condition Prediction Model** is a machine learning project that predicts a person's health condition using various health and lifestyle-related features. The project follows a complete machine learning pipeline, including data preprocessing, handling class imbalance, model training, evaluation, and prediction.

---

## ✨ Features

* Data loading and exploration
* Missing value handling
* Outlier detection and treatment using the IQR method
* Categorical feature encoding
* Numerical feature scaling
* Training and testing data split
* Class imbalance detection and balancing using undersampling
* Training multiple machine learning models
* Model comparison and performance evaluation
* Cross-validation for model validation
* Feature importance visualization
* Real-time health condition prediction
* Model saving and loading

---

## 📂 Project Workflow

1. Load the dataset.
2. Perform exploratory data analysis (EDA).
3. Handle missing values.
4. Treat outliers using the IQR method.
5. Encode categorical variables.
6. Scale numerical features.
7. Split the dataset into training and testing sets.
8. Balance the imbalanced target classes using undersampling.
9. Train multiple machine learning models.
10. Evaluate model performance.
11. Select the best-performing model (XGBoost).
12. Visualize feature importance.
13. Predict health conditions using user input.
14. Save and reload the trained model.

---

## 🤖 Machine Learning Models

* Naive Bayes
* Logistic Regression
* Decision Tree
* Random Forest
* XGBoost Classifier ✅ (Best Performing Model)

---

## 📊 Evaluation Metrics

The models are evaluated using:

* Accuracy Score
* Classification Report
* Confusion Matrix
* 5-Fold Cross Validation

---

## 🏆 Best Model

After comparing all models, **XGBoost Classifier** achieved the best overall performance and was selected as the final model for prediction.
